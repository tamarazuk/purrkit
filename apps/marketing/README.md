# PurrKit Marketing Site

Landing page for PurrKit.

## Development

From the repo root:

```bash
pnpm --filter @purrkit/marketing dev
```

The app runs at [http://localhost:3030](http://localhost:3030).

## Scripts

```bash
pnpm --filter @purrkit/marketing lint
pnpm --filter @purrkit/marketing check-types
pnpm --filter @purrkit/marketing build
pnpm --filter @purrkit/marketing start
```

## Configuration

- `NEXT_PUBLIC_SITE_URL` controls metadata and OG URLs.
